Introduction

  • There's no need to use multiple style files to make changes on page direction all you need to use ( dir ) attribute at your html tag
  • materialize dynamic based on materializecss so be sure you will write materializecss classes and methods
  • our new design system comes with some new features like:
    • outline iconic font
    • pre-built font family for rtl & ltr languages ( you can use both together )
      • Noto Naskh Arabic ( خط نسخ عربي افتراضي )
      • Raleway

Helpers

  • .left: class for floating left
  • .right: class for floating right
  • .float-dynamic: class for floating left if document direction is left & floating right if document direction is right
  • .float-dynamic-reverse: class for floating left if document direction is right & floating right if document direction is left
  • .align-left: class for left align
  • .right-align: class for right align
